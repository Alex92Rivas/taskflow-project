## Prompts utilizados en el proyecto

### Prompt 1 — Definir rol
Prompt:
Actúa como un desarrollador senior de JavaScript y revisa este código buscando posibles mejoras.

Por qué funciona:
Definir un rol hace que la IA responda con un enfoque más profesional y analítico.

---

### Prompt 2 — Explicación de código
Prompt:
Explica paso a paso qué hace esta función JavaScript.

Por qué funciona:
Pedir razonamiento paso a paso ayuda a entender mejor cómo funciona el código.

---

### Prompt 3 — Refactorización
Prompt:
Refactoriza esta función JavaScript para hacerla más limpia y fácil de mantener.

Por qué funciona:
La IA intenta mejorar la estructura del código sin cambiar su comportamiento.

---

### Prompt 4 — Generar función
Prompt:
Escribe una función JavaScript que seleccione un elemento aleatorio de un array.

Por qué funciona:
El prompt describe claramente la tarea que debe realizar el código.

---

### Prompt 5 — Validación de formulario
Prompt:
Genera una función JavaScript que valide un campo de texto antes de enviarlo.

Por qué funciona:
El prompt especifica claramente el objetivo y el tipo de código esperado.

---

### Prompt 6 — Buscar errores
Prompt:
Analiza este código JavaScript e identifica posibles errores o malas prácticas.

Por qué funciona:
La IA se centra en revisar el código buscando problemas.

---

### Prompt 7 — Explicación sencilla
Prompt:
Explica este código JavaScript como si estuviera aprendiendo programación.

Por qué funciona:
Hace que la explicación sea más simple y fácil de entender.

---

### Prompt 8 — Generar comentarios
Prompt:
Añade comentarios JSDoc a las funciones de este archivo JavaScript.

Por qué funciona:
La IA genera documentación clara para cada función.

---

### Prompt 9 — Optimización
Prompt:
Optimiza este código JavaScript para mejorar su rendimiento.

Por qué funciona:
La IA analiza el código buscando mejoras en eficiencia.

---

### Prompt 10 — Nuevas funcionalidades
Prompt:
Sugiere nuevas funcionalidades para mejorar una aplicación web de lista de películas.

Por qué funciona:
La IA puede proponer ideas creativas para ampliar el proyecto.
