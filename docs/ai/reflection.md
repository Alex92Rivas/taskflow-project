# Reflexión final sobre el uso de IA en el desarrollo

Durante el desarrollo de este proyecto utilicé herramientas de inteligencia artificial para asistir en diferentes fases del trabajo. Estas herramientas me ayudaron tanto en la escritura de código como en la revisión, documentación y mejora de algunas partes del proyecto.

Uno de los primeros ejercicios consistió en comparar distintos asistentes de IA, principalmente ChatGPT y Claude. Ambos modelos fueron capaces de identificar errores en funciones JavaScript y sugerir correcciones. En general, los dos ofrecieron respuestas correctas, aunque en algunos casos uno explicaba el problema con mayor claridad que el otro. Esto me permitió entender mejor cómo diferentes asistentes pueden abordar el mismo problema de formas distintas.

También utilicé Cursor como entorno de desarrollo con inteligencia artificial integrada. Esta herramienta facilita mucho el trabajo porque permite interactuar directamente con el código del proyecto. Por ejemplo, es posible pedir explicaciones sobre funciones concretas, solicitar refactorizaciones o generar mejoras en el código sin salir del editor. Esto agiliza bastante el flujo de trabajo.

Otro aspecto interesante fue el uso de servidores MCP (Model Context Protocol). Mediante esta configuración el asistente puede acceder al sistema de archivos del proyecto y entender mejor su estructura. Esto permite realizar consultas más avanzadas, como analizar archivos del proyecto, buscar funciones concretas o explicar partes del código basándose en el contexto real del repositorio.

A lo largo del desarrollo también utilicé la IA para mejorar la calidad del código. Por ejemplo, se refactorizaron funciones, se añadieron comentarios y se reorganizó parte de la lógica para hacerla más clara. Además, la IA fue útil para generar documentación inicial que posteriormente revisé manualmente para asegurar que el contenido fuera correcto y comprensible.

Sin embargo, también es importante destacar que la IA no sustituye completamente al programador. En varias ocasiones fue necesario revisar las respuestas generadas, corregir pequeños errores o adaptar el código a las necesidades concretas del proyecto. Por tanto, el uso de IA funciona mejor como una herramienta de apoyo que como una solución automática.

En conclusión, la inteligencia artificial puede ser una herramienta muy útil para el desarrollo de software, especialmente para aprender, resolver dudas o mejorar el código existente. No obstante, sigue siendo fundamental comprender el funcionamiento del programa y revisar cuidadosamente las sugerencias generadas por estas herramientas.
